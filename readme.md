# PaceArena

PaceArena is a static landing page and brand prototype for a virtual racing platform where real-world runners compete in shared digital race environments.

The product concept combines GPS-validated outdoor runs, live or asynchronous race rooms, rankings, wearable data, anti-fraud checks, social reactions, AI race narration, and augmented-reality podium moments.

Landing-page Preview visit: https://eng-soft-41.github.io/pacearena/

## Current State

This repository currently contains a static HTML/CSS site and brand/design references. There is no package manager, build step, JavaScript framework, or backend service in this repo.

The main page is localized in Brazilian Portuguese.

## Preview Locally

You can open `index.html` directly in a browser.

For a local server, run one of these from the project root:

```sh
python3 -m http.server 8000
```

Then visit:

```text
http://localhost:8000
```
